# Comandos
1 **Comandos Git**
📑 `git --version`
`cd` me sirve para navegar entre Carpetas
`cd ..` para devolverme una Carpeta
`git init` para iniciar Repositorio vacío
`git status` para verificar el estado de Archivos
Set - ExecutionPolicy Unrestricted para habilitar ejecución de comandos
`git add .` para agregar los archivos a mi git (caja)
`gitt commit -m "first commit"` para sellar la caja
`git clone` para clonar Repositorio
